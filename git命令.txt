获取git仓库的方式：
1.将尚未进行版本控制的本地目录转换为git仓库
在目标目录内右键执行git bash here 
> 执行 git init命令将当前目录转换为git仓库（创建一个.git隐藏目录）
2.从其他服务器上clone一个git仓库
git clone +地址

git status   (-s)                           查看文件状态
git add 文件名   跟踪文件（暂存）           git add .    向暂存区添加多个文件
git commit -m "注释”             提交暂存文件到git 仓库中
git reset --hard HEAD~1          删除最后一次提交
git update-ref -d HEAD           删除命名引用HEAD的当前分支所以提交
git checkout - 文件名               撤销对文件的修改
git reset HEAD 文件名/ .           移除暂存文件(.表示全部）
git commit -a -m "注释"           将所有已跟踪的文件暂存并提交（跳过add暂存）
推送本地分支到git仓库     git push +git仓库地址  +分支
git rm -f 文件名                          移除文件（git仓库和工作区同时移除）
git rm --cached 文件名              只从git仓库移除文件

git remote -v                                      查看关联库
git remote rm  origin                          删除关联库
git remote add origin  + git库地址       添加关联库                  

查看分支：git branch -v   /git branch
创建并切换分支 ：checkout -b 分支名
创建分支 ： git branch 分支名
切换分支:  git checkout分支名
合并分支与当前分支：git merge 分支名

git log（原始版本用reflog) --pretty=online              查看历史版本
git reset --hard  版本号              回退到版本x
